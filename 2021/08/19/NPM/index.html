NPM

包管理器

1、简介

官方网站：https://www.npmjs.com/
NPM全称Node Package Manager，是Node.js包管理工具，是全球最大的模块生态系统，里面所有的模块都是开源免费的；也是Node.js的包管理工具，相当于前端的Maven 。

2、使用npm管理项目

1、创建文件夹npm

2、项目初始化

2.1 快速安装

npm install XXX 或者 npm i XXX

2.2 安装模块的位置

node_modules

使用

const mysql=require(“mysql”);

2.3 模块和package.json有什么关系

通过npm install xxx 会记录到package.json这个文件中

类似maven porn.xml 一个含义，记录作用：复用

1：package.json记录的依赖模块

2：通过npm install可以直接把package.json记录的依赖的模块全部下载下来

可以避免重复下载，很方便集成第三方模块

为什么比直接拷贝node_modules(文件太大)

2.4 如果安装模块很慢

npm install -g cnpm –registry=https://registry.npm.taobao.org

cnpm -v

遇到问题

cnpm : 无法加载文件 C:\Users\ASUS\AppData\Roaming\npm\cnpm.ps1，因为在此系统上禁止运行脚本

解决方法

2.5 下载多个

cnpm install jquery vue redis mysql

2.6下载指定的版本号

npm install xxx@版本号

2.7 卸载模板

npm uninstall vue
